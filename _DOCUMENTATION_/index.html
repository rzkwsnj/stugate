A) First Step - top

Stugate makes it easy for you to carry out validation activities in the form of administration, as well as payments for academic scenarios during training or exams, and is also flexible to be used in other scenarios, such as courses, events, seminars, etc. It offers more than 6 screens and all are fully designed & functional also easy to customize. This application has a code scanner feature that you can use for any code format such as 1D (Barcode, UPC, CODE_128), 2D (QR code) etc.

This application is a native android application using IDE android studio version 2021.2.1, you can download via this page. To get started, make sure the Android studio IDE is installed, and open the STUGATE -> CODE -> MOBILE_APP -> Stugate folder.

Open the IDE android studio, and select "Open an existing Android Studio Project" and select the Stugate folder to open the project.

After the Android Studio IDE loads the project, wait until the build process is complete.

Make sure the build process is complete, paying attention to all the green checklist marks, and you are ready to browse the code.

B) Configurations - top

This application has a file and folder structure according to the function in the project, if you have a little experience with the Android project, then this will make it easier for you to make the necessary changes.

For example, if you need to change the logo, and the image assets associated with the application, you can open the Stugate -> app -> src -> main -> res -> drawable folder.
